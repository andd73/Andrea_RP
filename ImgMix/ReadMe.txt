ImgMix di Andrea Dotta 
scritto in C#
05-04-2020
ver. 1.0
Programma per la classificazione di immagini. 
Funzioni del programma:
- carica le immagini filtrandole per .jpg e verificando che siano valide
- affianca ad ogni immagine un file txt con le coordinate delle etichette che si usano
- permette di editare aggiungere e rimuovere le etichette anche dopo
- restituisce un file .zip con le informazioni da passare a YOLO per la creazione
	del file modello della rete.
